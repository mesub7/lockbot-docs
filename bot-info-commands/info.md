# Info

## Info Command

The bot info command displays information about the bot itself.

{% hint style="success" %}
**Format: \[prefix\]info**

**User permissions required: None**

**Bot permissions required: None**â€“ _It's advised that LockBot should have the administrator permission to carry out functions correctly._
{% endhint %}

### This includes the following bot information:

* Bot developer
* Bot version
* Total users
* Total guilds
* Bot uptime \(shard\)
* Bot ping \(shard\)
* Bot operating system
* Bot discord.js version
* Bot shard
* Support server link

{% hint style="info" %}
This command will display given info on each run of the command. To see up to date info, run the command regularly.
{% endhint %}
